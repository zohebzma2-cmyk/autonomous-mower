# Demo-day go/no-go — showing the machine to a stranger, safely

Derived from BUILD.md §11; ALL boxes or no demo.

## Before anyone arrives
- [ ] Blades PHYSICALLY disconnected unless the demo is on the demo field with spectators behind the line
- [ ] E-stop test: press → ignition dead, bars neutral, PTO relay open (meter, not vibes)
- [ ] RC kill tested from the actual spectator distance
- [ ] Geofence drawn INSIDE the physical field margin by ≥3 m; breach-hold demonstrated once
- [ ] Battery + fuel checked; tyre pressures at nominal (TPMS green)
- [ ] Spectator line marked ≥15 m; one person = one job (operator holds the RC, nobody else)

## During
- [ ] Operator thumb on RC kill the entire run
- [ ] No spectator inside the fence, ever, including you
- [ ] First run = the taught path they watched you teach; no improvising missions live

## Abort criteria (any one = end the demo, no debate)
RTK drops below FIXED · any unexpected motion · anyone crosses the line · wind-blown debris on field · you feel rushed.
